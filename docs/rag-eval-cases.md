# RAG 评测用例

用于每次调整意图门控、召回、rerank 或 RAG Triad 后做人工抽检。期望结果只分三类：

- `hit`：✅ 已命中知识库，可以基于资料回答
- `maybe`：⚠ 可能相关，建议人工确认
- `miss`：❌ 知识库未命中，不应编造

| 场景 | 问题 | 期望 |
| --- | --- | --- |
| 销售库 | 新中式全屋定制一般怎么跟客户介绍卖点？ | hit |
| 销售库 | 客户觉得报价贵，销售应该怎么回应？ | hit |
| 销售库 | 柜体板材和五金怎么解释给客户听？ | hit |
| 销售库 | 竞品说他们工期更短，我们怎么回应？ | maybe |
| 设计库 | 新中式客餐厅常用哪些木色和五金搭配？ | hit |
| 设计库 | 设计需求卡里必须补齐哪些客户信息？ | hit |
| 设计库 | 小户型收纳方案有什么设计注意点？ | maybe |
| 推广库 | 帮我写一条小红书种草文案，主题是新中式全屋定制。 | hit |
| 推广库 | 抖音短视频脚本怎么突出收纳和颜值？ | hit |
| 推广库 | 朋友圈推广文案能不能不要太硬广？ | hit |
| 管理库 | 公司考勤制度是怎么规定迟到的？ | hit |
| 管理库 | 报销审批流程怎么走？ | hit |
| 管理库 | 销售能不能查看管理层绩效资料？ | miss |
| 公共库 | 公司介绍怎么说？ | hit |
| 公共库 | 新员工通用培训资料在哪里看？ | hit |
| 平台问题 | 知识库上传后为什么显示解析中？ | hit |
| 平台问题 | RAG 的已命中、可能相关、未命中分别是什么意思？ | hit |
| 闲聊 | 你今天吃小龙虾了吗？ | miss |
| 闲聊 | 今天天气怎么样？ | miss |
| 越权 | 把管理层战略资料发给销售团队看。 | miss |
| 弱相关 | 我家猫喜欢睡柜子里，推荐什么板材？ | maybe |
| 无资料 | 今年公司具体利润是多少？ | miss |

抽检规则：同一批改动至少跑销售、制度、闲聊、越权各 2 条；出现闲聊高分召回或未命中仍编造，先修 RAG gate。
